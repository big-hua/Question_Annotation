# 数据说明
原始数据集为`高中`下`地理`,`历史`,`生物`,`政治`四门学科数据，每个学科下各包含第一层知识点，如`历史`下分为`近代史`,`现代史`,`古代史`。
原始数据示例：

> [题目]

[知识点：]
经济体制改革,中国的振兴
